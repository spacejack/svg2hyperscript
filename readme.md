# svg2hyperscript

## Install

    npm install svg2hyperscript

## Usage

```javascript
const svg2h = require('svg2hyperscript')
const hyperscriptSrc = svg2h(svgSrc, 'h')
```

## CLI Usage

    svg2h svgfile [-o outfile.js] [-fn hyperscriptFunction]
